Short description about the application

The study aimed to develop a mobile application that monitors basketball players' shooting percentages and identifies missed shots easily and accurately. It includes proper visualization of game outputs per quarter with touch coordinates saved in the database. This tool helps coaches, trainers, and players analyze strengths and weaknesses, improve game progress, and provide insights into individual and team performance.
